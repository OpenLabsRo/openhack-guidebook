# Submissions

The project submission platform is available at [**app.openhack.ro**](https://app.openhack.ro/).

### **Access & Setup**

- Log in using the **same email address** used for event registration.
- Verify that your **team information, names, and other details** are correct.
- If any information appears incorrect, contact an **event admin** as soon as possible to resolve it.

---

### **Registration Requirement**

All participants are required to **complete their registration** on the submission platform in order to **check in at the event**.

Please ensure your information is filled in **before arriving on Saturday**.

---

### **Web App Installation**

For an improved experience, you can install the platform as a web app:

**Menu (⋮) → Install App**
